# Trace

A calm knowledge app built around thoughts, not folders.

Write things down. Connect ideas. Organize later.

<br>

## Why Trace?

Folders become rigid over time.  
Pure graph tools become overwhelming.

Trace stays out of your way, virtual folders let you organize without moving files, one note can live in multiple places at once, and nothing ever breaks when you rename something.

Structure should support thinking, not control it.

<br>

## Features

- **Virtual folders**: organize without touching the filesystem
- **Linked notes**: `[[wiki links]]` that resolve automatically
- **Local-first**: your files, your disk, plain Markdown
- **Fast**: built in Rust; large vaults stay snappy
- **Distraction-free**: nothing you don't need

<br>

## Philosophy

Write first.  
Organize later.  
Leave a trace.
